# Project Overview

## About the project

- Build an agent that lets a non-technical user ask questions in plain english about data in the database.
- The agent should translate the question into SQL, execute it, return the answer in natural language, and visualize it with a chart where appropriate.
- Present this through a UI of your choice (a simple chat interface is fine).
- Use input data of your choice, but the database must have a minimum of 4 tables that require joins to answer questions.

---

## Knowledge to test

• Engineering & UX: schema handling, chart-type selection, latency
• Eval: SQL correctness, answer groundedness, ambiguous query handling
• Guardrails & security: SQL injection, prompt injection, read-only enforcement, destructive queries

---

## Demo Guideline

- No authentication or no production deployment is required.
- Use synthetic data where appropriate.
- Git repo or ZIP file of the working solution with all necessary files.
- A README.md covering setup and run instructions
- A demo video of 5 minutes max, walking through the solution.
- Values: clear thinking and structured communication.
